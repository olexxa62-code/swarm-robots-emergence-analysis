# Contributing to A.2 Swarm Robots Kappa Analysis

## How to Contribute

This project is part of the emergence parameter κ ≈ 1 research framework. Contributions should maintain scientific rigor and academic standards.

### Reporting Issues

- Use clear, descriptive titles
- Include system information and error messages
- Provide minimal reproducible examples
- Reference specific equation or figure numbers when applicable

### Code Contributions

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/description`)
3. Follow existing code structure and style
4. Add proper attribution headers to all new files
5. Ensure all calculations are verified
6. Update documentation accordingly
7. Submit a pull request with clear description

### Standards

- **Language**: Academic English only
- **Code Style**: PEP 8 for Python
- **Documentation**: Markdown with proper citations
- **Attribution**: All files must include full attribution header
- **Testing**: Verify κ calculations before submission
- **Figures**: 600 DPI for publication quality

### Attribution Format

All source files must include:
```
System Classification: A.2 swarm_robots_kappa_analysis
Author: [Your Name]
Developer: [Organization]
Theoretical Framework: The Emergence Parameter κ ≈ 1: An Empirical Signature of Criticality in Physical and Biological Systems
```

### Review Process

All contributions undergo:
1. Mathematical verification
2. Code quality review
3. Documentation completeness check
4. Attribution compliance verification
